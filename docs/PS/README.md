# PS快捷键
- 文件--打开 --  可以打开 我们要测量的图片
- ctrl+r 可以打开标尺  或者  视图 --  标尺
- 右击标尺，  把里面的单位改为  像素  
- ctrl+ 加号 键  可以 放大  视图  ctrl+ 减号 缩小视图
- 按住空格键，  鼠标可以 变成小手 ，拖动 ps 视图
- 用选区 拖动  可以 测量 大小 
- ctrl+ d  可以取消选区  或者旁边空白处点击一下也可以取消选区


# PS切图
## ps切图片，分两大步：
**1:用切片选中图片**<br>
* 利用切片工具手动划出

* 图层菜单---新建基于图层的切片

* 利用辅助线 来切图 --    基于参考线的切片 <br>
**2:导出切片**<br>
 文件菜单   -- 存储为web设备所用格式 ----   选择  我们要的图片格式 ----  点存储  ---   别忘了选中的切片
